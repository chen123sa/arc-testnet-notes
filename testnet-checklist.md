# Arc Testnet Checklist

- [ ] Create dedicated EVM testnet wallet
- [ ] Add Arc Testnet RPC

- [ ] Install Foundry
- [ ] Initialize `hello-arc`
- [ ] Add `ARC_TESTNET_RPC_URL=https://rpc.testnet.arc.network`
- [ ] Create `HelloArchitect.sol`
- [ ] Run `forge test`
- [ ] Run `forge build`
- [ ] Deploy to Arc Testnet if wallet is funded
- [ ] Record tx hash / contract address / explorer URL
