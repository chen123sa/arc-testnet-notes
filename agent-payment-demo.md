# AI Agent Payment Workflow Demo

## Concept

A lightweight demo showing how an AI agent could coordinate a task and settle payment using Arc testnet.

## Flow

1. User creates a task.
2. Agent estimates scope/cost.
3. Payment request is generated in testnet USDC.
4. Arc testnet transaction records settlement.
5. Agent marks task as paid/completed.
6. Demo shows transaction/explorer reference.

## Why this fits Arc

Arc docs highlight agentic economy and USDC-native settlement. This demo connects both themes without pretending to be a production financial app.
