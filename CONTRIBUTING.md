# Contributing

This is a personal Arc public testnet learning repo. Contributions are welcome if they keep the scope practical and reproducible.

Useful contributions:

- Clear bug reports with steps to reproduce
- Improvements to Foundry scripts/tests
- Better Arc onboarding notes
- Small demo improvements related to AI-agent settlement flows
- Documentation fixes

Please avoid:

- Token speculation
- Airdrop/reward farming content
- Mainnet/private-key examples
- Unverified claims about Arc production behavior

## Safety

Never submit private keys, seed phrases, API keys, or real-funds wallet data.
